# 编辑服务详细设计文档

## 1. 模块概述

### 1.1 功能定位

编辑服务提供视频预览和简单剪辑功能，允许用户在最终输出前对视频进行调整。这是一个低优先级的辅助功能，主要用于满足用户对结果微调的需求。

### 1.2 核心职责

- **视频预览**：提供视频播放和字幕预览功能
- **简单剪辑**：支持视频片段裁剪
- **多段切分**：将长视频切分为多个独立片段
- **字幕编辑**：在线编辑字幕内容和时间轴

### 1.3 设计原则

由于本模块优先级较低，设计遵循以下原则：
- 功能简洁，不追求专业剪辑软件的复杂功能
- 复用现有组件，减少开发成本
- 基于 FFmpeg 实现，保持技术栈统一

## 2. 业务流程

### 2.1 视频预览流程

```
用户选择视频
    │
    ▼
加载视频元数据
    │
    ▼
加载关联字幕文件
    │
    ▼
前端播放器渲染
    │
    ▼
同步显示字幕
```

### 2.2 视频剪辑流程

```
用户设置剪辑参数
    │
    ├── 起始时间
    ├── 结束时间
    └── 输出配置
    │
    ▼
提交剪辑任务
    │
    ▼
调用 FFmpeg 执行剪辑
    │
    ▼
生成新视频文件
    │
    ▼
更新数据库记录
```

### 2.3 多段切分流程

```
用户定义切分点
    │
    ▼
验证切分点有效性
    │
    ▼
生成切分任务列表
    │
    ▼
依次执行 FFmpeg 切分
    │
    ▼
生成多个视频片段
    │
    ▼
创建片段关联记录
```

## 3. 视频预览

### 3.1 预览方式

| 方式 | 说明 | 适用场景 |
|------|------|----------|
| 原生播放 | 直接播放原视频文件 | 本地部署，文件较小 |
| 流式播放 | 后端转码为流媒体格式 | 大文件，远程访问 |

### 3.2 字幕同步

预览时支持同步显示字幕：
- 加载 SRT 字幕文件
- 根据播放时间匹配当前字幕
- 支持切换原始字幕和翻译字幕
- 支持双字幕同时显示

### 3.3 预览接口

后端提供视频文件流式读取接口，支持：
- Range 请求（支持拖动进度条）
- 字幕文件读取
- 视频元数据查询

## 4. 简单剪辑

### 4.1 剪辑功能

| 功能 | 说明 |
|------|------|
| 片段裁剪 | 指定起止时间，提取视频片段 |
| 首尾裁剪 | 去除视频开头或结尾部分 |

### 4.2 FFmpeg 剪辑

使用 FFmpeg 的 `-ss` 和 `-to` 参数进行精确剪辑。

**剪辑模式**：
- 快速剪辑：不重新编码，速度快但可能不精确
- 精确剪辑：重新编码，精确到帧但速度较慢

### 4.3 字幕同步处理

剪辑视频时需要同步处理字幕：
- 根据剪辑时间调整字幕时间轴
- 移除不在剪辑范围内的字幕条目
- 生成新的字幕文件

### 4.4 输出选项

| 选项 | 说明 |
|------|------|
| 覆盖原文件 | 替换原视频（危险操作，需确认） |
| 新建文件 | 生成新文件，保留原文件 |
| 自定义命名 | 用户指定输出文件名 |

## 5. 多段切分

### 5.1 切分方式

| 方式 | 说明 |
|------|------|
| 手动切分 | 用户指定每个切分点的时间 |
| 等分切分 | 按指定数量或时长等分 |
| 按章节切分 | 根据视频章节信息自动切分 |

### 5.2 切分点定义

用户可通过以下方式定义切分点：
- 在预览界面点击添加切分点
- 输入具体时间戳列表
- 设置等分参数（段数或每段时长）

### 5.3 输出管理

切分后生成多个视频片段：
- 文件命名：`原文件名_part01.mp4`、`原文件名_part02.mp4` 等
- 每个片段生成对应的字幕文件
- 在数据库中建立片段与原视频的关联关系

### 5.4 批量处理

切分任务作为批量任务处理：
- 每个片段作为一个子任务
- 支持并行处理多个片段
- 整体进度 = 已完成片段数 / 总片段数

## 6. 字幕编辑

### 6.1 编辑功能

| 功能 | 说明 |
|------|------|
| 文本编辑 | 修改字幕文字内容 |
| 时间轴调整 | 修改字幕起止时间 |
| 条目增删 | 添加或删除字幕条目 |
| 整体偏移 | 批量调整所有字幕的时间偏移 |

### 6.2 编辑界面

前端提供字幕编辑界面：
- 左侧：视频预览窗口
- 右侧：字幕列表，可编辑
- 点击字幕条目跳转到对应时间点
- 实时预览编辑效果

### 6.3 保存机制

- 自动保存草稿（本地存储）
- 手动保存到服务器
- 保存时验证字幕格式有效性
- 支持导出为 SRT 文件

## 7. 进度计算

### 7.1 剪辑任务进度

与视频合成服务相同，解析 FFmpeg 输出计算进度：

```
进度 = (已处理时长 / 目标时长) × 100
```

### 7.2 切分任务进度

```
进度 = (已完成片段数 / 总片段数) × 100
```

### 7.3 上报内容

进度百分比、当前操作、已处理时长、预计剩余时间。

## 8. 配置项

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| default_clip_mode | string | fast | 默认剪辑模式（fast/precise） |
| max_split_segments | int | 20 | 最大切分段数 |
| min_segment_duration | int | 10 | 最小片段时长（秒） |
| auto_save_interval | int | 30 | 字幕自动保存间隔（秒） |
| preview_quality | string | medium | 预览质量（low/medium/high） |

## 9. 错误处理

### 9.1 常见错误

| 错误类型 | 原因 | 处理方式 |
|----------|------|----------|
| 视频文件不存在 | 文件被删除 | 不可重试 |
| 时间参数无效 | 起始时间大于结束时间 | 参数校验拒绝 |
| FFmpeg 执行失败 | 命令错误或资源不足 | 检查日志 |
| 磁盘空间不足 | 存储空间不够 | 不可重试，告警 |
| 字幕格式错误 | 编辑后格式不合法 | 提示用户修正 |

### 9.2 错误恢复策略

- FFmpeg 失败：检查错误日志，简化参数重试
- 切分中断：已完成的片段保留，从断点继续
- 字幕保存失败：保留本地草稿，提示用户

## 10. 接口设计

### 10.1 服务接口

| 方法 | 说明 |
|------|------|
| execute(task, progress_callback) | 执行编辑任务 |
| cancel(task_id) | 取消任务 |
| clip_video(video_id, start, end, config) | 剪辑视频 |
| split_video(video_id, split_points, config) | 切分视频 |
| get_preview_url(video_id) | 获取预览地址 |
| save_subtitle(video_id, subtitle_type, content) | 保存字幕 |

### 10.2 任务 Payload

| 字段 | 说明 |
|------|------|
| video_id | 视频 UUID |
| operation | 操作类型（clip/split） |
| start_time | 起始时间（剪辑） |
| end_time | 结束时间（剪辑） |
| split_points | 切分点列表（切分） |
| clip_mode | 剪辑模式 |
| output_name | 输出文件名 |

### 10.3 任务 Result

| 字段 | 说明 |
|------|------|
| video_id | 原视频 UUID |
| output_files | 输出文件列表 |
| subtitle_files | 对应字幕文件列表 |
| segments_info | 片段信息（切分时） |

- 工作进程：调用执行编辑任务
