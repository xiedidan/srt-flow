# 素材生成服务详细设计文档

## 1. 模块概述

### 1.1 功能定位

素材生成服务负责基于视频内容和字幕自动生成发布所需的素材，包括标题、摘要和缩略图。它封装了 LLM API 调用和图像处理逻辑，为视频二次发布提供便捷的素材准备功能。

### 1.2 核心职责

- **LLM 调用封装**：统一封装多种 LLM API，提供文本生成能力
- **标题生成**：基于视频内容生成多个候选标题
- **摘要生成**：基于字幕内容生成视频摘要
- **缩略图生成**：从视频截取关键帧或调用绘图接口生成

### 1.3 生成模式

用户可勾选任意素材组合进行生成，默认全选：

| 素材类型 | 默认 | 说明 |
|----------|------|------|
| 标题 | ✓ | 生成标题候选 |
| 摘要 | ✓ | 生成视频摘要 |
| 缩略图 | ✓ | 生成缩略图 |

用户可根据需要任意勾选一个或多个素材类型。

## 2. 业务流程

### 2.1 素材生成流程

```
接收素材生成任务
    │
    ▼
检查输入文件是否存在
    │
    ├── 字幕文件不存在 ──→ 返回错误
    │
    ▼
读取视频元数据和字幕内容
    │
    ▼
┌─────────────────────────────────┐
│  根据配置执行生成任务           │
│    │                           │
│    ├── 标题生成 ──→ 调用 LLM   │
│    │                           │
│    ├── 摘要生成 ──→ 调用 LLM   │
│    │                           │
│    └── 缩略图生成 ──→ 截帧/绘图 │
└─────────────────────────────────┘
    │
    ▼
保存生成结果到 assets 目录
    │
    ▼
更新数据库记录
    │
    ▼
返回生成结果
```

### 2.2 输出文件

素材生成完成后在视频目录的 `assets/` 子目录生成：

| 文件 | 说明 |
|------|------|
| title_candidates.txt | 标题候选列表 |
| summary.txt | 视频摘要 |
| thumbnail.jpg | 缩略图 |

## 3. LLM 调用封装

### 3.1 独立的 LLM 实例配置

素材生成服务使用独立的 LLM 实例配置，与翻译服务分开管理。原因：

1. **温度参数差异**：素材生成属于创作任务，需要较高的温度参数（0.7-0.9）以获得更有创意的输出；而翻译任务需要较低温度（0.3-0.5）保证准确性
2. **服务类型不同**：素材生成包含图片生成服务，需要单独配置绘图 API

**便捷功能**：支持从翻译服务一键复制 LLM 实例配置，用户可在此基础上调整参数。

### 3.2 实例配置项

每个 LLM 实例包含以下配置：

| 配置项 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| name | string | 是 | 实例名称 |
| provider | string | 是 | 提供商类型：deepseek/gemini/openai |
| api_key | string | 是 | API Key（加密存储） |
| base_url | string | 是 | API 地址 |
| model | string | 是 | 模型名称 |
| temperature | float | 否 | 温度参数，默认 0.8（创作任务较高） |
| enabled | bool | 否 | 是否启用，默认 true |
| priority | int | 否 | 优先级，默认 100 |

### 3.3 调用参数

| 参数 | 说明 |
|------|------|
| model | 使用实例配置的模型 |
| temperature | 标题生成 0.8-0.9，摘要生成 0.6-0.7 |
| max_tokens | 根据任务类型动态设置 |

### 3.4 故障转移

- 单次调用失败时在当前实例重试
- 重试失败后按优先级尝试下一个实例
- 所有实例失败则任务失败

## 4. 标题生成

### 4.1 生成策略

基于以下信息生成标题候选：
- 原始视频标题
- 字幕内容（摘要或关键句）
- 视频元数据（频道、分类等）
- 目标平台特性

### 4.2 Prompt 设计要点

Prompt 包含以下要素：
- 角色设定：视频标题创作专家
- 原始标题和视频内容摘要
- 目标平台（YouTube、Bilibili 等）
- 生成要求：吸引眼球、包含关键词、符合平台风格
- 输出格式：多个候选标题，按推荐度排序

### 4.3 输出格式

生成 5-10 个候选标题，每行一个，保存到 `title_candidates.txt`。

### 4.4 配置项

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| candidate_count | int | 5 | 生成候选数量 |
| target_platform | string | general | 目标平台 |
| include_emoji | bool | false | 是否包含表情符号 |
| max_length | int | 80 | 标题最大长度 |

## 5. 摘要生成

### 5.1 生成策略

基于字幕内容生成结构化摘要：
- 提取视频主题和核心观点
- 按时间线或逻辑结构组织
- 控制摘要长度

### 5.2 字幕预处理

由于字幕可能很长，采用首尾摘取策略：

**摘取规则**：
- 开头部分：从第一条字幕开始后 30 秒内的所有字幕
- 结尾部分：最后一条字幕结束前 30 秒内的所有字幕
- 如果视频较短（总时长 < 90 秒），则使用全部字幕

**设计理由**：
- 视频开头通常包含主题介绍和核心观点
- 视频结尾通常包含总结和结论
- 中间部分多为详细展开，对摘要贡献较小
- 大幅减少 token 消耗，提高处理效率

### 5.3 Prompt 设计要点

Prompt 包含以下要素：
- 角色设定：内容摘要专家
- 视频标题
- 字幕内容（需说明：提供的是视频开头和结尾部分的字幕）
- 摘要要求：基于开头和结尾内容推断视频主题，生成准确、简洁的摘要
- 输出格式：纯文本或带章节标题

### 5.4 输出格式

生成的摘要保存到 `summary.txt`，支持两种格式：
- 纯文本：一段连续的摘要文字
- 结构化：带章节标题的分段摘要

### 5.5 配置项

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| summary_style | string | plain | 摘要风格（plain/structured） |
| max_length | int | 500 | 摘要最大字数 |
| include_timestamps | bool | false | 是否包含时间戳 |

## 6. 缩略图生成

### 6.1 生成方式

支持两种缩略图生成方式：

| 方式 | 说明 | 适用场景 |
|------|------|----------|
| 视频截帧 | 从视频中截取关键帧 | 快速生成，保持原视频风格 |
| AI 绘图 | 调用绘图 API 生成 | 需要创意图片 |

### 6.2 视频截帧

使用 FFmpeg 从视频中截取帧作为缩略图。

**截帧策略**：
- 指定时间点：用户指定具体时间
- 智能选取：分析视频选择画面丰富的帧
- 多帧候选：截取多个时间点供用户选择

**截帧参数**：
- 输出尺寸：默认 1920x1080（1080P）
- 输出格式：JPEG
- 质量：可配置压缩质量

### 6.3 AI 绘图

调用绘图 API 生成缩略图，流程：
1. 基于视频内容生成绘图提示词（Prompt）
2. 调用绘图 API 生成图片
3. 下载并保存图片

**支持的绘图服务**：
- DALL-E（OpenAI）
- Midjourney API
- Stable Diffusion API
- 其他兼容 API

### 6.4 绘图提示词生成

使用 LLM 基于视频内容生成绘图提示词：
- 分析视频主题和风格
- 生成描述性的英文提示词
- 包含风格、构图、色调等要素

### 6.5 配置项

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| generation_mode | string | frame | 生成方式（frame/ai） |
| frame_time | string | auto | 截帧时间（auto/具体时间） |
| frame_count | int | 1 | 截帧数量 |
| output_width | int | 1920 | 输出宽度 |
| output_height | int | 1080 | 输出高度 |
| image_quality | int | 90 | JPEG 质量（1-100） |
| ai_provider | string | dall-e | AI 绘图服务 |
| ai_style | string | realistic | 绘图风格 |

## 8. 配置项汇总

### 8.1 全局配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| generate_title | bool | true | 是否生成标题 |
| generate_summary | bool | true | 是否生成摘要 |
| generate_thumbnail | bool | true | 是否生成缩略图 |
| use_translated_subtitle | bool | true | 使用翻译后字幕 |
| summary_head_seconds | int | 30 | 摘要摘取开头秒数 |

### 8.2 子任务 AI 配置

每个子任务有独立的 AI 服务商和模型配置：

#### 标题生成 AI 配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| title_ai_provider_id | string | null | AI 服务商 ID |
| title_model_name | string | deepseek-chat | 模型名称 |
| title_temperature | float | 0.85 | 温度参数（创作性较高） |
| title_system_prompt | string | null | 自定义系统提示词 |
| title_user_prompt_template | string | null | 自定义用户提示词模板 |

#### 摘要生成 AI 配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| summary_ai_provider_id | string | null | AI 服务商 ID |
| summary_model_name | string | deepseek-chat | 模型名称 |
| summary_temperature | float | 0.65 | 温度参数（准确性较高） |
| summary_system_prompt | string | null | 自定义系统提示词 |
| summary_user_prompt_template | string | null | 自定义用户提示词模板 |

#### 缩略图生成 AI 配置（AI 模式）

缩略图 AI 生成需要两类配置：
1. **文本模型配置**：用于根据视频标题和摘要生成文生图提示词
2. **图像生成 API 配置**：用于调用文生图服务生成缩略图

##### 文本模型配置（生成提示词）

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| thumbnail_ai_provider_id | string | null | 文本模型 AI 服务商 ID |
| thumbnail_model_name | string | deepseek-chat | 文本模型名称 |
| thumbnail_temperature | float | 0.7 | 温度参数 |
| thumbnail_system_prompt | string | null | 自定义系统提示词 |
| thumbnail_user_prompt_template | string | null | 自定义用户提示词模板 |

##### 图像生成 API 配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| thumbnail_image_provider | string | dall-e | 图像生成服务商 |
| thumbnail_image_api_key | string | null | API Key（加密存储） |
| thumbnail_image_base_url | string | null | API 地址 |
| thumbnail_image_model | string | dall-e-3 | 图像生成模型 |

**设计理由**：
- 三个子任务的创作性质不同，需要不同的温度参数
- 用户可能希望使用不同的模型（如标题用 GPT-4，摘要用 DeepSeek）
- 缩略图 AI 生成需要先用文本模型生成提示词，再调用图像生成 API

### 8.3 绘图服务配置

| 配置项 | 类型 | 说明 |
|--------|------|------|
| provider | string | 绘图服务提供商 |
| api_key | string | API Key（加密存储） |
| base_url | string | API 地址 |

## 9. 错误处理

### 9.1 常见错误

| 错误类型 | 原因 | 处理方式 |
|----------|------|----------|
| 字幕文件不存在 | 未完成翻译或识别 | 不可重试 |
| LLM 调用失败 | 网络错误或服务不可用 | 尝试下一个实例 |
| 绘图 API 失败 | 服务不可用或配额用尽 | 回退到视频截帧 |
| FFmpeg 截帧失败 | 视频文件损坏 | 不可重试 |
| 内容过长 | 字幕超过 token 限制 | 分段处理 |

### 9.2 错误恢复策略

- LLM 调用失败：按优先级尝试下一个实例
- 绘图 API 失败：可配置自动回退到视频截帧
- 部分生成失败：保存已成功的部分，记录失败项

## 10. 接口设计

### 10.1 服务接口

| 方法 | 说明 |
|------|------|
| execute(task, progress_callback) | 执行素材生成任务 |
| cancel(task_id) | 取消生成 |
| can_retry(error) | 判断错误是否可重试 |
| generate_titles(content, config) | 生成标题候选 |
| generate_summary(content, config) | 生成摘要 |
| generate_thumbnail(video_path, config) | 生成缩略图 |

### 10.2 任务 Payload

| 字段 | 说明 |
|------|------|
| video_id | 视频 UUID |
| generate_title | 是否生成标题 |
| generate_summary | 是否生成摘要 |
| generate_thumbnail | 是否生成缩略图 |
| subtitle_path | 字幕文件路径 |
| title_config | 标题生成配置 |
| summary_config | 摘要生成配置 |
| thumbnail_config | 缩略图生成配置 |

### 10.3 任务 Result

| 字段 | 说明 |
|------|------|
| video_id | 视频 UUID |
| title_candidates_path | 标题候选文件路径 |
| summary_path | 摘要文件路径 |
| thumbnail_path | 缩略图文件路径 |

## 11. 依赖关系

素材生成服务依赖：
- 配置管理器：读取生成配置和 API Key
- 数据库管理器：更新视频记录
- 日志管理器：记录生成日志
- 翻译服务：可选复制 LLM 实例配置，使用翻译后的字幕
- 外部工具：FFmpeg（截帧）
- 外部服务：LLM API、绘图 API

被以下模块依赖：
- 工作进程：调用执行素材生成任务
