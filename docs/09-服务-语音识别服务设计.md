# 语音识别服务详细设计文档

## 1. 模块概述

### 1.1 功能定位

语音识别服务负责从视频中提取音频并生成字幕文件，是 SRT Flow 处理流水线的核心环节。它封装了 Whisper 本地模型和在线大模型 API，提供统一的语音识别接口。

### 1.2 核心职责

- **音频提取**：从视频文件中提取音频轨道
- **语音识别**：调用 Whisper 或在线模型进行语音转文字
- **字幕生成**：生成标准 SRT 格式字幕文件
- **时间戳对齐**：确保字幕时间戳与音频精确对齐
- **进度上报**：实时报告识别进度

### 1.3 识别通道

系统提供两种识别通道，由用户选择：

| 通道 | 说明 | 适用场景 |
|------|------|----------|
| Whisper 本地 | 本地部署的 Whisper 模型 | 隐私要求高、批量处理 |
| 在线大模型 | Gemini 等在线 API | 快速处理、无 GPU 环境 |

## 2. 业务流程

### 2.1 识别流程

```
接收识别任务
    │
    ▼
检查视频文件是否存在
    │
    ├── 不存在 ──→ 返回错误
    │
    ▼
提取音频（如果不存在）
    │
    ▼
根据配置选择识别通道
    │
    ├── Whisper 本地 ──→ 加载模型 → 执行推理
    │
    └── 在线大模型 ──→ 上传音频 → 调用 API
    │
    ▼
解析识别结果
    │
    ▼
生成 SRT 字幕文件
    │
    ▼
更新数据库记录
    │
    ▼
返回识别结果
```

### 2.2 输出文件

识别完成后在视频目录生成：

- `audio_original.m4a`：提取的音频文件
- `subtitle_original.srt`：识别生成的字幕

---

## 3. 音频提取

### 3.1 提取方式

使用 FFmpeg 从视频中提取音频轨道，默认输出 M4A 格式（AAC 编码），比特率 128k。

### 3.2 音频预处理

针对 Whisper 识别进行优化处理：

- 转换为单声道（如果是立体声）
- 采样率转换为 16kHz（Whisper 推荐）
- 音量归一化（可选）

---

## 4. Whisper 本地识别

### 4.1 引擎类型

系统支持两种 Whisper 本地引擎：

| 引擎 | 说明 | 适用场景 |
|------|------|----------|
| whisper_local | 使用 faster-whisper Python 库 | 有 Python 环境，需要灵活配置 |
| faster_whisper_xxl | 独立可执行文件 | 无需 Python 环境，开箱即用 |

### 4.2 模型管理

支持的模型规格：

| 模型 | 参数量 | VRAM 需求 | 相对速度 | 准确度 |
|------|--------|-----------|----------|--------|
| tiny | 39M | ~1GB | 最快 | 较低 |
| base | 74M | ~1GB | 快 | 一般 |
| small | 244M | ~2GB | 中等 | 良好 |
| medium | 769M | ~5GB | 较慢 | 较高 |
| large-v3 | 1550M | ~10GB | 慢 | 最高 |

默认使用 large-v3 模型以获得最佳字幕质量。

**模型下载与管理：**

- 通过 Web 界面下载和删除模型
- 模型文件跨平台共享，从 Hugging Face 下载 faster-whisper 模型
- 两种本地引擎都需要下载平台特定的可执行文件（Windows/Linux/macOS）
- 文件存储在 `data/models/` 目录
- 支持下载进度显示

**模型加载策略：**
- 首次使用时需手动下载模型
- 采用单例模式避免重复加载
- 支持配置自定义模型存储路径

### 4.3 推荐实现

推荐使用 faster-whisper 库，相比原版 Whisper 速度提升 4 倍，内存占用更低，支持 CTranslate2 优化。

### 4.4 GPU 加速

- 默认使用 CUDA 加速
- 自动检测 GPU 可用性，无 GPU 时回退到 CPU
- 支持配置指定 GPU 设备 ID

## 5. 在线大模型识别

### 5.1 支持的服务

| 服务 | 说明 |
|------|------|
| Google Gemini | 支持音频直接输入，多语言识别 |
| OpenAI Whisper API | 云端 Whisper 服务，支持直接输出 SRT |

### 5.2 调用流程

1. 上传音频文件到服务端
2. 发送识别请求，指定输出格式
3. 解析返回的文本和时间戳
4. 转换为标准 SRT 格式

### 5.3 音频分片

在线 API 通常有文件大小限制（如 OpenAI 限制 25MB）。超过限制时：

1. 按时长分片（如每 10 分钟一片）
2. 分片识别后合并结果
3. 调整后续分片的时间戳偏移

---

## 6. SRT 字幕生成

### 6.1 SRT 格式规范

标准 SRT 格式包含：序号、时间轴（HH:MM:SS,mmm --> HH:MM:SS,mmm）、字幕文本，各条目以空行分隔。

### 6.2 字幕分段优化

Whisper 输出的分段可能过长或过短，需要按以下规则优化：

| 规则 | 说明 |
|------|------|
| 最大字符数 | 单条字幕不超过 80 字符 |
| 最大时长 | 单条字幕不超过 7 秒 |
| 最小时长 | 单条字幕不少于 1 秒 |
| 句子边界 | 优先在句号、问号等标点处分段 |

分段算法：获取原始分段后，检查是否符合规则，过长则按标点或字数拆分，过短则与相邻段合并，最后重新计算时间戳。

### 6.3 时间戳处理

- 保留毫秒精度
- 确保相邻字幕不重叠
- 添加适当间隔（如 100ms）

| 配置项 | 类型 | 默认值 | 说明 |

| whisper_model | string | large-v3 | Whisper 模型大小 |
| whisper_device | string | cuda | 运行设备 |

| whisper_language | string | auto | 源语言（auto 自动检测） |

| online_provider | string | gemini | 在线服务提供商 |

| openai_api_key | string | null | OpenAI API Key（加密） |
| max_subtitle_chars | int | 80 | 单条字幕最大字符数 |
| max_subtitle_duration | float | 7.0 | 单条字幕最大时长（秒） |

## 8. 错误处理

### 8.1 常见错误

| 错误类型 | 原因 | 处理方式 |
|----------|------|----------|
| 视频文件不存在 | 文件被删除或路径错误 | 不可重试 |
| 音频提取失败 | FFmpeg 错误 | 检查视频格式 |
| 模型加载失败 | 内存不足或模型损坏 | 尝试更小模型 |
| GPU 内存不足 | 显存不够 | 任务失败，提示用户选择更小模型 |
| API 调用失败 | 网络错误或配额用尽 | 可重试 |
| 识别结果为空 | 音频无语音内容 | 返回空字幕 |

### 8.2 错误恢复策略

- 网络错误：自动重试，最多 3 次
- GPU 内存不足：任务失败，提示用户选择更小的模型或释放显存
- 模型加载失败：尝试重新下载模型
- 在线 API 失败：可重试，不自动切换到本地 Whisper

## 9. 进度上报

### 9.1 进度计算

Whisper 本地识别：基于音频时长和已处理时长计算，每处理一个分段更新进度。

在线 API 识别：上传阶段占 0-20%，识别阶段占 20-90%，后处理阶段占 90-100%。

### 9.2 上报内容

进度百分比、当前阶段、已处理时长、总时长、状态消息。

## 10. 接口设计

### 10.1 服务接口

| 方法 | 说明 |
|------|------|
| execute(task, progress_callback) | 执行识别任务 |
| cancel(task_id) | 取消识别 |
| can_retry(error) | 判断错误是否可重试 |
| extract_audio(video_path, output_path) | 提取音频 |
| transcribe_local(audio_path, config) | 本地 Whisper 识别 |
| transcribe_online(audio_path, config) | 在线 API 识别 |
| generate_srt(segments, output_path) | 生成 SRT 文件 |

### 10.2 任务 Payload

| 字段 | 说明 |
|------|------|
| video_id | 视频 UUID |
| video_path | 视频文件路径 |
| channel | 识别通道（whisper_local/online） |
| language | 源语言（auto 自动检测） |
| model | Whisper 模型大小 |

### 10.3 任务 Result

| 字段 | 说明 |
|------|------|
| video_id | 视频 UUID |
| audio_path | 提取的音频文件路径 |
| subtitle_path | 生成的字幕文件路径 |
| language_detected | 检测到的语言 |
| duration | 音频时长（秒） |
| segments_count | 字幕条数 |

## 11. 依赖关系

语音识别服务依赖：
- 配置管理器：读取识别配置和 API Key
- 数据库管理器：更新视频记录
- 日志管理器：记录识别日志
- 外部工具：FFmpeg、faster-whisper

被以下模块依赖：
- 工作进程：调用执行识别任务
- 翻译服务：使用生成的字幕文件
