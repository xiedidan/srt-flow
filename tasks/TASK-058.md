# TASK-058: 语音合成固定角色/音色

## 任务信息
- 优先级: 高
- 状态: Review
- 来源: 问题日记
- 相关功能: 系统配置-语音合成

## 问题描述

语音合成需要锁定一个角色，避免每一句语音都用不同的角色，导致声音不一致。

## 修复要求

1. **角色锁定方案**
   - 方案一：直接锁定固定角色/声音
   - 方案二：使用参考音色（0 Shot 音色迁移，如果模型支持）

2. **配置支持**
   - 在语音合成配置中增加角色/音色锁定选项
   - 支持选择已有角色或上传参考音频

## 技术要点

- 与 TASK-054（参考音频管理）配合使用
- 不同 TTS 模型的角色锁定方式可能不同
- 需要在任务创建时指定使用的角色/音色

## 依赖任务

- TASK-054: 实现语音合成参考音频管理

## 实现方案

### 前端修改 (TtsTaskCreateModal.vue)

1. 添加参考音频选择功能
   - 加载已上传的参考音频列表
   - 显示支持语音克隆的引擎标识
   - 提供参考音频选择 UI（单选列表）
   - 支持试听参考音频

2. 任务创建时传递参考音频信息
   - `reference_audio_id`: 选中的参考音频 ID
   - `speaker`: 当选择参考音频时，设置为 `ref:{audio_id}` 格式

### 后端修改

1. `backend/app/main.py`
   - TTS 任务预处理时解析 `reference_audio_id`
   - 通过 `ReferenceAudioRepository` 获取音频文件路径
   - 将文件路径注入到 `task.payload["speaker"]`

2. `backend/core/repositories.py`
   - 新增 `ReferenceAudioRepository` 类
   - 提供 `get_by_id` 方法获取参考音频记录

### 支持语音克隆的引擎

- IndexTTS: 零样本克隆，需要参考音频
- SparkTTS: 支持参考音频
- CozyVoice: 支持零样本克隆模式
- Coqui TTS: 支持参考音频

### 不支持语音克隆的引擎

- ChatTTS: 使用内置音色
- VITS: 使用预训练音色

## 验收标准

- [x] 可以为语音合成任务指定固定角色/音色
- [x] 整个视频的语音合成使用一致的声音
- [x] 支持参考音频方式的音色迁移（如模型支持）
