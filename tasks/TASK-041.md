# 任务描述

实现翻译批次重叠功能，让翻译批次之间衔接更为流畅。每个批次前后 N 行的内容也一并提交给 AI 作为上下文参考，但只需要翻译批次本身的内容。

# 需求来源

docs/需求日记.md - 翻译批次重叠

# 功能要点

1. 配置管理中增加"批次重叠行数"默认值配置
2. 新建翻译任务时，允许用户设置本次任务的重叠行数
3. 翻译服务在处理每个批次时，将前后 N 行内容作为上下文一并提交
4. 提示词中明确指示 AI 只翻译批次本身的内容，上下文仅供参考

# 验收标准 (Acceptance Criteria)

- [x] 全局配置增加"翻译批次重叠行数"配置项，默认值可配置
- [x] 翻译任务创建表单增加"重叠行数"输入字段
- [x] 后端翻译服务支持批次重叠逻辑
- [x] 提示词正确区分上下文内容和待翻译内容
- [x] 更新配置说明文档
- [x] 更新 API 文档
