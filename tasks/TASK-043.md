# TASK-043: 实现语音合成任务页面

## 任务描述

完善语音合成（TTS）任务管理页面，包括任务创建弹窗、任务列表展示、任务详情查看等功能。

## 相关文档

- docs/11-服务-语音合成服务设计.md

## 功能需求

### 1. TTS 任务创建弹窗 (TtsTaskCreateModal.vue)

- 视频选择：从已完成翻译的视频中选择
- TTS 引擎选择：Coqui TTS / ChatTTS / SparkTTS / IndexTTS / CozyVoice / VITS
- 说话人/音色选择（根据引擎动态加载）
- 输出格式选择：M4A / WAV / MP3
- 高级选项：
  - 语速范围设置（0.5 - 2.0）
  - 是否启用时间拉伸
  - GPU 加速开关
- 批量创建支持（多选视频）

### 2. 任务列表增强

- 支持卡片/列表视图切换
- 状态筛选（全部/进行中/已完成/失败）
- 标签筛选
- 显示使用的 TTS 引擎信息

### 3. 任务详情展示

- 显示合成配置（引擎、音色、语速等）
- 显示进度（已合成字幕数/总字幕数）
- 完成后显示输出音频信息（时长、格式、大小）
- 支持试听合成的音频

### 4. 音频预览功能

- 完成的任务支持在线试听
- 显示音频波形（可选）

## 技术要点

- 复用 AsrTasksView 的视图切换和筛选逻辑
- 创建 TtsTaskCreateModal 组件
- 添加音频播放器组件或复用现有组件

## 验收标准

- [ ] 可以创建 TTS 任务并选择引擎和配置
- [ ] 任务列表正确显示 TTS 任务
- [ ] 支持视图切换和状态筛选
- [ ] 任务详情显示完整配置信息
- [ ] 完成的任务可以试听音频
