# TASK-056: 字幕翻译预览叠加字幕

## 任务信息
- 优先级: 高
- 状态: Review
- 来源: 问题日记
- 相关功能: 系统配置-字幕翻译-字幕预览

## 问题描述

字幕翻译完成后的预览功能，视频播放时没有叠加显示字幕。

## 修复要求

预览视频时，需要将翻译后的字幕叠加显示在视频上，方便用户检查翻译结果。

## 技术要点

- 复用 TASK-055 的字幕叠加实现
- 支持显示原始字幕和翻译字幕（双语显示）
- 字幕需要与视频时间轴同步

## 依赖任务

- TASK-055: 语音识别预览叠加字幕

## 修复内容

修改 `frontend/src/components/asr/SubtitleEditorModal.vue`：

1. 新增 `originalSubtitles` 响应式变量，存储原始字幕用于双语显示
2. 新增 `isTranslationTask` 计算属性，判断是否为翻译任务
3. 新增 `availableSubtitleModes` 计算属性，根据任务类型提供不同的字幕模式选项：
   - ASR 任务：显示/隐藏
   - 翻译任务：双语/译文/原文/隐藏
4. 修改 `playerSubtitles` 计算属性，为翻译任务合并原始字幕和翻译字幕
5. 新增 `findMatchingOriginal` 函数，通过时间范围匹配原始字幕（处理翻译后字幕合并导致的索引不一致）
6. 修改 `loadSubtitles` 函数，翻译任务时同时加载原始字幕
7. 更新模板，使用 v-for 动态渲染字幕模式切换按钮

修改 `frontend/src/components/preview/VideoPlayer.vue`：

1. 修复字幕叠加条件，当 `subtitleMode === 'none'` 时隐藏字幕

## 验收标准

- [x] 字幕翻译预览时字幕正确叠加在视频上
- [x] 支持双语字幕显示（原文+译文）
- [x] 字幕与视频时间轴同步
- [x] 字幕样式清晰可读
- [x] 用户可以切换字幕显示模式（双语/译文/原文/隐藏）
