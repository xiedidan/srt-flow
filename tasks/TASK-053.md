# 任务描述
增加快速配置的引导功能，通过一个流程界面来引导用户一步步完成系统相关配置。在系统首次访问的时候自动弹出配置引导界面，用户也可以从首页按需点击进入（对于已经配置好的功能则加载相关配置，可供编辑）。

# 验收标准 (Acceptance Criteria)
- [ ] 实现配置引导向导组件，支持分步骤配置
- [ ] 系统首次访问时自动弹出配置引导
- [ ] 首页提供"快速配置"入口按钮
- [ ] 引导流程覆盖主要配置项（AI服务商、下载、语音识别、翻译、语音合成、视频合成）
- [ ] 已配置的功能显示当前配置，支持编辑
- [ ] 记录用户是否已完成首次配置引导
