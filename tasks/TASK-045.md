# TASK-045: 实现素材生成任务页面

## 任务描述

完善素材生成任务管理页面，包括任务创建弹窗、生成选项配置、结果展示等功能。

## 相关文档

- docs/13-服务-素材生成服务设计.md

## 功能需求

### 1. 素材生成任务创建弹窗 (AssetTaskCreateModal.vue)

- 视频选择：从已有字幕的视频中选择
- 生成项目勾选：
  - [ ] 标题候选（默认勾选）
  - [ ] 视频摘要（默认勾选）
  - [ ] 缩略图（默认勾选）
- 标题生成配置：
  - 候选数量（5-10）
  - 目标平台（通用/YouTube/Bilibili）
  - 是否包含表情符号
- 摘要生成配置：
  - 摘要风格（纯文本/结构化）
  - 最大字数
- 缩略图生成配置：
  - 生成方式（视频截帧/AI 绘图）
  - 截帧时间（自动/指定时间）
  - 输出尺寸
- 批量创建支持

### 2. LLM 实例配置（可选，在设置页面）

- 独立的素材生成 LLM 实例配置
- 支持从翻译服务复制配置
- 温度参数调整（创作任务较高）

### 3. 任务列表增强

- 支持卡片/列表视图切换
- 状态筛选
- 显示生成项目信息

### 4. 任务详情与结果展示

- 显示生成配置
- 显示进度（当前阶段）
- 完成后展示生成结果：
  - 标题候选列表（可复制）
  - 摘要内容（可复制）
  - 缩略图预览（可下载）

### 5. 结果操作

- 标题：一键复制
- 摘要：一键复制、编辑后保存
- 缩略图：预览、下载

## 技术要点

- 创建 AssetTaskCreateModal 组件
- 创建 AssetResultView 组件展示生成结果
- 图片预览和下载功能

## 验收标准

- [x] 可以创建素材生成任务并选择生成项目
- [x] 各项生成配置可以自定义
- [x] 任务列表正确显示素材生成任务
- [x] 完成后可以查看和操作生成结果
- [x] 标题和摘要支持复制
- [x] 缩略图支持预览和下载
