# TASK-044: 实现视频合成任务页面

## 任务描述

完善视频合成任务管理页面，包括任务创建弹窗、字幕样式配置、任务列表展示等功能。

## 相关文档

- docs/12-服务-视频合成服务设计.md

## 功能需求

### 1. 视频合成任务创建弹窗 (SynthesisTaskCreateModal.vue)

- 视频选择：从已有字幕的视频中选择
- 字幕模式选择：
  - 仅原始字幕
  - 仅翻译字幕
  - 双字幕叠加
- 视频编码选择：H.264 / H.265 / AV1
- 编码质量设置（CRF 值）
- 硬件加速选项（CUDA / QSV / 无）
- 批量创建支持

### 2. 字幕样式配置组件 (SubtitleStyleConfig.vue)

- 主字幕样式配置：
  - 字体、字号、颜色
  - 边框宽度和颜色
  - 阴影效果
  - 位置（上/中/下）
  - 背景色和透明度
- 副字幕样式配置（双字幕模式）
- 样式预设选择（default / youtube / netflix / minimal 等）
- 实时预览效果（可选）

### 3. 任务列表增强

- 支持卡片/列表视图切换
- 状态筛选
- 显示字幕模式和编码格式信息

### 4. 任务详情展示

- 显示合成配置（字幕模式、编码、样式等）
- 显示进度（已处理时长/总时长）
- 完成后显示输出视频信息（分辨率、大小、编码）
- 支持预览合成的视频

## 技术要点

- 创建 SynthesisTaskCreateModal 组件
- 创建 SubtitleStyleConfig 组件用于样式配置
- 复用 VideoPlayer 组件进行视频预览

## 验收标准

- [ ] 可以创建视频合成任务并配置字幕模式
- [ ] 字幕样式可以自定义配置
- [ ] 支持选择样式预设
- [ ] 任务列表正确显示合成任务
- [ ] 完成的任务可以预览视频
