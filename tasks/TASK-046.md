# TASK-046: 实现视频剪辑任务页面

## 任务描述

完善视频剪辑任务管理页面，包括剪辑任务创建、多段切分、字幕编辑等功能。

## 相关文档

- docs/14-服务-编辑服务设计.md

## 功能需求

### 1. 剪辑任务创建弹窗 (EditorTaskCreateModal.vue)

- 视频选择
- 操作类型选择：
  - 片段裁剪：设置起止时间
  - 多段切分：定义切分点
- 片段裁剪配置：
  - 起始时间输入
  - 结束时间输入
  - 剪辑模式（快速/精确）
- 多段切分配置：
  - 手动添加切分点
  - 等分切分（按段数或时长）
- 输出选项：
  - 新建文件（默认）
  - 自定义文件名

### 2. 时间轴选择器组件 (TimelineSelector.vue)

- 可视化时间轴
- 拖动选择起止时间
- 点击添加切分点
- 显示当前选择范围
- 与视频预览联动

### 3. 字幕在线编辑功能

- 复用或增强现有 SubtitleEditor 组件
- 支持编辑字幕文本
- 支持调整时间轴
- 支持添加/删除字幕条目
- 整体时间偏移调整
- 保存到服务器

### 4. 任务列表增强

- 支持卡片/列表视图切换
- 状态筛选
- 显示操作类型（裁剪/切分）

### 5. 任务详情展示

- 显示剪辑配置
- 显示进度
- 切分任务显示各片段状态
- 完成后显示输出文件列表

## 技术要点

- 创建 EditorTaskCreateModal 组件
- 创建 TimelineSelector 组件
- 增强字幕编辑功能
- 复用 VideoPlayer 组件

## 验收标准

- [ ] 可以创建片段裁剪任务
- [ ] 可以创建多段切分任务
- [ ] 时间轴选择器可以直观操作
- [ ] 字幕可以在线编辑和保存
- [ ] 任务列表正确显示剪辑任务
- [ ] 完成后可以查看输出文件

## 优先级

低（编辑服务本身优先级较低）
